# 宠物王国工具

一款专为宠物王国及类似系列单机游戏打造的数据加解密与存档修改工具。支持对游戏内的 `json`、`png` 及存档数据文件进行处理。

## ✨ 主要功能

- **🛠 多格式支持**：完美支持 `json`、`png` 及存档数据文件的加解密。
- **🔍 自动识别**：自动识别文件类型，无需手动配置。
- **📄 单文件处理**：快速处理单个文件，操作简单便捷。
- **📦 批量文件处理**：支持批量文件操作，提升工作效率。
- **🪟 悬浮窗处理**：内置悬浮窗功能，随时随地进行数据处理。
- **✏️ JSON 文本修改**：支持直接编辑 JSON 文本内容。

## 🎮 支持的游戏

目前已支持该系列游戏及其衍生作品，包括但不限于：

- 宠物王国单机版
- 口袋妖怪单机版
- 宠物王国外传
- 斗罗大陆单机版

## 📱 系统兼容

- **支持系统**：Android 7.0 ~ 16
